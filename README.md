# MLOps Assignment 2

## Project Overview
This repository contains the modularized scripts for fine-tuning a BERT model for Goodreads review classification, tracking experiments with W&B, and deploying to Hugging Face Hub.

## Task 2: Modular Scripts
- `data.py`: Data loading and preprocessing.
- `train.py`: Model training with W&B integration.
- `eval.py`: Model evaluation.
- `utils.py`: Helper classes and metrics.
- `deploy.py`: Script to push the model to Hugging Face.
